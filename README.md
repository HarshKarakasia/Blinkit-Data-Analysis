# Blinkit Data Analysis – Power BI Project

## Overview
This project presents a detailed data analysis of **Blinkit**, an instant-delivery platform, using **Power BI**.  
The dashboard provides insights into sales performance, customer behavior, product categories, and operational metrics.

This repository contains:
- `Blinkit data analysis.pbix` → Power BI dashboard
- `README.md` → Project documentation

---

## Objectives
- Analyze overall sales and revenue trends  
- Identify top-performing product categories  
- Study customer ordering patterns  
- Understand region-wise performance  
- Evaluate operational efficiency and ratings  
- Provide insights to improve business KPIs  

---

## Tools & Technologies
- **Power BI Desktop**  
- Power Query  
- DAX (Data Analysis Expressions)  
- CSV/Excel Dataset  
- GitHub for version control  

---

## Dataset
- **Source:** Kaggle / Provided Dataset  
- **Format:** CSV/Excel  
- **Columns include:**
  - Item Fat Content  
  - Item Type  
  - Outlet Size  
  - Outlet Location Type  
  - Outlet Establishment Year  
  - Item Visibility  
  - MRP / Sales  
  - Rating  
  - And other retail metrics
